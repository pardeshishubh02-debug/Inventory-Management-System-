# Inventory Management System (N-Tier Architecture)

This project is an **Inventory Management System** developed in C# using N-Tier Architecture.  
It demonstrates a clean separation of concerns by dividing the application into MODEL, DAL, BAL, and UI layers.

The solution follows **N-Tier Architecture**:

- **MODEL**  
  Contains entity classes and data models.

- **DAL (Data Access Layer)**  
  Handles all database operations such as Insert, Update, Delete, and Select.

- **BAL (Business Access Layer)**  
  Contains business logic and validations. Acts as a bridge between UI and DAL.

- **InventoryManagementTask (UI Layer)**  
  The main application layer that interacts with users.
